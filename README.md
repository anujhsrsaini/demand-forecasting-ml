# Demand Forecasting ML

**Author:** Anuj Saini

A machine learning project designed to forecast product demand and optimize inventory levels.

## Project Structure
- `sql/`: SQL scripts for data extraction, cleaning, and preprocessing.
- `findings/`: Key findings, reports, and analytical insights.
- `notebooks/`: Jupyter notebooks for exploratory data analysis (EDA), model training, and evaluation.
- `dashboards/`: Dashboard configurations, designs, and visualizations.
